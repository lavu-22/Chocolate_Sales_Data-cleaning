# Chocolate Sales Data Cleaning

## Objective
The objective of this task is to clean and preprocess the Chocolate Sales dataset using Excel.

## Dataset
Chocolate Sales Dataset (Kaggle)

## Steps Performed
- Checked for missing values.
- Checked for duplicate records.
- Converted the Amount column to Number format.
- Verified the Date column format.
- Reviewed the dataset for consistency.

## Tools Used
- Microsoft Excel

## Result
The dataset was cleaned and prepared for analysis.
